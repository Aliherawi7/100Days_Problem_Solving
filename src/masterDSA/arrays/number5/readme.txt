

### 5. Find Missing and Doubled Numbers In Array

descriptions:
Given an unsorted array of size n. array elements are
in the range of 1 to n. One number from set {1,2, ...n}
is missing and one number occurs twice in the array.
find these two numbers.

Example 1:
input = [1,2,3,5,6,6}
output = missing number = 4, and twice occurred number = 6
