

### 50. Generate number by length and

descriptions:
write a program to that get two number m and n as parameter and generate the maximum
and minimum number that has the m digits and sum of its digits is equal to n.
if there was not any number with this features then return -1 -1

note:
0 <= m <= 300
1 <= n <= 900


Example 1:
Input: 2 and 15
Output: min = 69 and max = 96

Example 2:
Input: 2 and 15
